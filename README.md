# 🪞 Smart Mirror

A **Smart Mirror** is an IoT-based project developed using **Raspberry Pi** and **Python** that transforms a regular mirror into an interactive smart display. The mirror provides useful real-time information such as the current time, date, live weather updates, and personalized greetings, making daily information easily accessible without requiring additional devices.

This project was built to explore the integration of **IoT hardware, Python programming, APIs, and real-time data visualization**. It provided practical experience in connecting software with hardware while creating a user-friendly smart display system.

---

# 📖 Project Overview

Traditional mirrors only serve the purpose of reflection, whereas a Smart Mirror combines a reflective surface with a digital display to provide useful information in real time. The objective of this project was to develop a cost-effective smart mirror capable of displaying dynamic information using Raspberry Pi.

The application continuously updates information such as time, weather conditions, and greetings, providing users with an enhanced and interactive experience.

# ⚙️ How It Works

### Step 1
The Raspberry Pi powers the Smart Mirror application when the system starts.

### Step 2
Python continuously updates the current date and time.

### Step 3
The application connects to the weather API to retrieve live weather information.

### Step 4
Based on the current time, the mirror displays personalized greetings such as:

- Good Morning
- Good Afternoon
- Good Evening

### Step 5
The interface automatically refreshes the displayed information after a fixed interval.
